This project develops an end-to-end Business Intelligence (BI) solution for a chain of specialized supply stores offering tools, hydraulic equipment, trailer parts, agricultural products, and workshop supplies.

The solution combines web scraping, data transformation, and BI visualization to analyze product pricing, customer perception, and engagement, supporting data-driven commercial and operational decisions.

The dataset is collected through web scraping using Selenium and Python, transformed into analytics-ready datasets, and visualized using Power BI.
